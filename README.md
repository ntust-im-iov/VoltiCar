# VoltiCar

## 專案簡介

VoltiCar 是一個專注於電動車充電站管理的 Flutter 應用程式。這個應用程式幫助用戶輕鬆找到附近的充電站，並提供即時的充電站狀態資訊。本專案旨在解決電動車用戶在尋找充電站時遇到的困擾，提供一個直觀且實用的解決方案。

## 目前功能

### 已完成功能
- 基本的地圖顯示功能
- 充電站位置標記
- 簡單的用戶介面

### 開發中功能
- 充電站詳細資訊顯示
- 路線規劃
- 充電站狀態即時更新

## 技術架構

### 前端
- Flutter/Dart
- Google Maps API
- Provider 狀態管理

### 後端
- Firebase
  - Authentication
  - Firestore
  - Cloud Functions
